# rollcall-activity
Used in a class activity to help students become familiar with GitHub workflows.

# Instructions
1. Log into GitHub
2. Fork [this repository](https://github.com/bepalaciosa/rollcall-activity)
3. In your forked copy, add your information to rollcall.md
4. Issue a pull request back to [this repository](https://github.com/bepalaciosa/rollcall-activity)
5. Report back to the class worksheet.
